1.1. 便利なPythonを導入する（for win）¶

Pythonで科学技術計算を行うには、 IPhython Notebook が使いやすい。

使い方

  1. Python(x,y) をダウンロードしてインストール
  2. コマンドプロンプトで、ipython notebook –pylab inlineを入力すれば起動。
  3. 上記のコマンドをbatファイルに書き込めば簡単に起動が可能。

python(x,y)は2.7系

1.3.1. リスト¶

大かっこ[]でくくったオブジェクト群の事

リスト作成の例

>>> L = [1, 2, 3, 4]
>>> s = ['red', 'green', 'yellow']
>>> L[1]
2
>>> s[2]
'yellow'

:リテラルを使うことで自由にリストから値を取り出せます。次の例では2つ目の要素からすべて取り出します

>>> L = [1, 2, 3, 4]
>>> L[1:]
[2, 3, 4]

次の例では2,3要素を取り出します。1:3の後半の要素は含みません。

>>> L = [1, 2, 3, 4]
>>> L[1:3]
[2, 3]

後ろからの要素も指定でします。-1番目一番最後の要素になります

>>> L = [1, 2, 3, 4]
>>> L[-4]
1
>>> L[1:-1]
[2, 3]

リストの中にリストを入れることもできますn 下の例は2行4列の配列を作成しています

>>> L = [[1, 2, 3, 4], [5, 6, 7, 8]]
>>> L[1]
[5, 6, 7, 8]
>>> L[1][2]
6

L配列の2列だけを取り出したいときは、リスト内包表記で取り出すのが便利です

>>> L = [[1, 2, 3, 4], [5, 6, 7, 8]]
>>> [ls[1] for ls in L]
[2, 6]

スマートでなくても、for文で記述しても構いません

>>> L2=[]
>>> for ls in L:
...    L1.append(ls[1])
...
>>> L2
[2, 6]

リストからの要素の取り出しは、データを扱うに当たって重要な操作になります。慣れておいたほうがよろしいかと思います

1.3.1.1. リストに関する重要な関数¶

list.append(x)¶

リストの末尾にxを追加します。

len(list)¶

リストの要素数を返します

list.insert(i, x)¶

指定した位置iにxを挿入します

list.remove(x)¶

リスト中でxの値を持つ最初の要素を削除します。

list.pop(i)¶
list.pop(list)

リストからi番目の要素を取り出し、削除します。引数としてリスト型の要素も使用できます。

list.index(x)¶

リストの中で値xを持っている最初の要素のインデックスを返す

list.count(x)¶

値xがリストに入っている個数を返す

list.sort()¶

リスト内の要素を昇順に並べる

list.reverse()¶

リスト内の要素を降順に並べる

関数の使用例

>>> L = [1, 2, 3, 4]
>>>#最後の要素に5を追加
>>>L.append(5)
>>>L
[1, 2, 3, 4, 5]
>>> #リストの最後に配列を追加
>>> L.extend([6, 7, 8])
>>> L
[1, 2, 3, 4, 5, 6, 7, 8]
>>>
