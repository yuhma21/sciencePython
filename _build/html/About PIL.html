5. About PIL¶

著者:yuhma21

5.1. PILモジュールについて¶

PIL(Python Imaging Library)はPythonで画像を処理するライブラリです。

5.2. PIL.Imageモジュール¶

PIL.Imageモジュールは、ファイルから画像を開く、新しい画像を作成するなどの様々な関数を提供している。

5.2.1. 関数¶

Image.new(mode, size)¶
Image.new(mode, size, color)

与えられたサイズで新しく画像を作成する。サイズは(幅, 高さ)のタプルで指定する。 色は、通常のHTML color nameで指定する。もしくは、rgb()関数を利用。 modeは”RGB”等を指定する

Modeの一覧

Mode 説明
1 1bit画像。黒か白
L 8bit画像。グレースケール
P 8bit画像。カラー
RGB 3x8bit画像。RGB
RGBA 4x8bit画像。RGB+Alpha(透明度)
CMYK 4x8bit画像。CMYK
YCbCr 3x8bit画像。ビデオフォーマット
I 32bit画像。整数型
F 32bit画像。浮動小数点型
PIL.Image.open(file)¶
PIL.Image.open(file, mode)

画像を開く。モードを指定するのであれば、必ず”r”を指定する。

Image.blend(image1, image2, alpha)¶

2つの画像をブレンドする。イメージとアルファは下記の関係

out = image1 * (1.0 - alpha) + image2 * alpha

サンプル画像のダウンロード ダウンロード , ダウンロード

from PIL import Image

im1 = Image.open('AquaTermi_lowcontrast.JPG')
im2 = Image.open('flower32_t0.png')
im2 = im2.resize(im1.size)

out = Image.blend(im1, im2, 0.5)

out.show()
Image.composite(image1, image2, mask)¶

image1を下地にimage2をかぶせる。ただし、maskのイメージを割合をAlpha値 にして、image2に透明度を加える。maskに利用する画像は、”1”、”L”、”RGBA” でなければならない。また、画像は全て同じサイズにする

サンプル画像のダウンロード ダウンロード , ダウンロード , ダウンロード

from PIL import Image

im1 = Image.open('AquaTermi_lowcontrast.JPG')
im2 = Image.open('flower32_t0.png')
im3 = Image.open("fisherman.jpg").convert("1")
im2 = im2.resize(im1.size)
im3 = im3.resize(im1.size)

out = Image.composite(im1, im2, im3)
out = out.resize((im1.size[0]/3, im1.size[1]/3))

out.show()
Image.eval(image, function)¶

imageの各ピクセルに対してfunctionの計算をかけて出力

#!/usr/bin/env python
# -*- coding: utf-8 -*-

from PIL import Image

#evalメソッド用の関数を用意する
#２値化反転する
def efunc(x):
    if x > 128:
        a = 0
    else:
        a = 255
    return a

im1 = Image.open('AquaTermi_lowcontrast.JPG').convert('L')

out = Image.eval(im1, efunc)

out.show()

5.2.2. メソッド¶

読み込んだイメージオブジェクトに対して利用するメソッド

im.convert(mode)¶
im.convert("P", **options)
im.convert(mode, matrix)

イメージオブジェクトを指定の方法で変換する

options パラメータ
dither= ディザリング。デフォルトはFLOYDSTEINBERG
palette= パレットの設定。デフォルトはWEB
colors= 使用する色数。
im.crop(box)¶

boxでタプル型の領域指定を行い、コピーする。(x1, y1, x2, y2)

im.filter(filter)¶

フィルタで得られた画像を返す。scipyのファイル他の方が良い？

im.histgram()¶
im.histgram(mask)

画像のヒストグラムをリストで返す。maskを利用すると0出ないところだけのヒストグラムを返す

im.paste(image, box)¶

コピーした画像をはりつける。boxは(x1, y1, x2, y2)を指定

im.resize(size)¶

sizeタプル(幅, 高さ)で指定した大きさに変更した画像を返す。

im.rotate(angle)¶

指定したangle(度)に時計回りに回転させる

サンプル画像のダウンロード ダウンロード

#!/usr/bin/env python
# -*- coding: utf-8 -*-

from PIL import Image

im = Image.open('flower32_t0.PNG')

out = im.rotate(45)

out.show()
im.save(outfile, options...)¶
im.save(outfile, format, options...)

ファイル名を指定して画像を保存する

im.show()¶

画像を表示する。表示の際は通常画像を開くプログラムで開く

im.thumbnail(size)¶

sizeタプルで指定した大きさのサムネイルを作成する

im.tobitmap()¶

画像をビットマップに変換して返す

transform(size, method, data)¶
transform(size, method, data, filter)

何やら変換して返す

5.2.3. 属性¶

im.format()¶

画像のフォーマットを返す

>>> from PIL import Image
>>> im = Image.open('foo.png')
>>> im.format
'PNG'
im.mode()¶

画像のmodeを返す。convertなどで指定するmodeの事。

>>> from PIL import Image
>>> im1 = Image.open('foo.png')
>>> im1.mode
'RGB'
>>> im2 = im1.convert('L')
>>> im2.mode
'L'
im.size()¶

画像のサイズをタプル(幅, 高さ)で返す
